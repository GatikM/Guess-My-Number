# Guess My Number

A fun and interactive number guessing game built with vanilla JavaScript, HTML, and CSS.

## Description

Guess My Number is a simple but engaging game where you try to guess a random number between 1 and 20. The game provides feedback on whether your guess is too high, too low, or correct. Test your luck and try to achieve the highest score!

## How to Play

1. Enter a number between 1 and 20 in the input field
2. Click the "Check!" button to submit your guess
3. The game will tell you if your guess is:
   - **Too High** (ğŸ“ˆ) - Your guess is greater than the secret number
   - **Too Low** (ğŸ“‰) - Your guess is less than the secret number
   - **Correct** (ğŸŠ) - You found the number!
4. Each wrong guess reduces your score by 1
5. Try to reach the highest score before running out of attempts
6. Click "Again!" to start a new game

## Game Rules

- You start with a score of 20
- Each incorrect guess costs you 1 point
- When your score reaches 0, the game is over
- Your highscore is saved and displayed throughout your session
- Click "Again!" to reset and play a new round
